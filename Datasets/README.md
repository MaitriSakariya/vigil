# Evaluation Dataset

This directory contains sample images used to validate the project's inference pipeline.

The images represent common real-world scenes such as streets, parking lots, office environments, and pedestrians.

These samples were used for functional testing of the application.

## Structure

- `images/` – Sample images used during evaluation.

## Evaluation

A small evaluation dataset is included under the `datasets/` directory. The project was validated using these sample images along with the automated test suite (`14/14` tests passing).
